Présentation

Deno est un environnement javascript basé sur le moteur v8 même moteur. que chromium ainsi que sur le langage Rust. Cet environnement à était crée par Ryand dahl. Il a pour but de régler certains soucis de nodeJs. Notamment en termes de Sécurité.

Installation

Deno est installable sur la plupart des machines (Linux,Windows,Macos).

Installation sous Windows avec powershell

iwr https://deno.land/x/install/install.ps1 -useb | iex

Installation sous Linux fonctionne aussi sous Mac

curl -fsSL https://deno.land/x/install/install.sh | sh

Installation sous mac avec Homebrew

brew install deno

Démonstration

A l'utilisation, Deno se différencie de node surtout par les permissions

La liste des permissions est :

  • --allow-write : Permission d'écrire
  • --allow-read : Permission de lire
  • --allow-net : Permission d'accés réseaux
  • --allow-env : Permission d'accès aux variables d'environement

Comparaison

Les deux ayant principalement les mêmes fonctionnalités. Il y a très peu de points qui les différe.
Mais Deno apporte plus de sécurité et aussi un meilleur support du TypeScript
