Présentation

Deno est un environnement javascript basé sur le moteur v8 même moteur que chromium ainsi que sur le langage Rust.
Cette environement à était crée par Ryand dahl. il a pour but de régler certains souci de nodeJs. Nottament en termes de
sécurité.

Instalation

Deno est installable sur la plupart des machines (Linux,Windows,Macos).

Installation sous Windows avec powershell

iwr https://deno.land/x/install/install.ps1 -useb | iex

Installation sous Linux fonctionne aussi sous Mac

curl -fsSL https://deno.land/x/install/install.sh | sh

Installation sous mac avec Homebrew

brew install deno

Démonstration

A l'utilisation Deno ce différencie de node surtout par les permissions

la liste des permission est :

  • --allow-write : Permission d'écrire
  • --allow-read : Permission de lire
  • --allow-net : Permission d'accés réseaux
  • --allow-env : Permission d'accès aux variables d'environement

Comparaison

Les deux ayants principalement les mêmes fonctionnalités. Il y'a très peux de points qui les différe
Mais deno apporte plus de securité et aussi un meilleur support du TypeScript
